# 🚀 Bhavesh Pund | Portfolio Frontend

A modern, responsive, and dynamic personal portfolio built using **React** and **Vite**. The application showcases my projects, technical skills, certifications, education, experience, and contact information through a clean and interactive user interface. It also includes a secure **Admin Dashboard** for managing portfolio content dynamically.

## 🌐 Live Demo

**Portfolio:**  
👉 https://bhaveshpund.vercel.app

---

# ✨ Features

- Responsive UI for Desktop, Tablet, and Mobile
- Modern animations with Framer Motion
- Dynamic content loaded from Spring Boot REST APIs
- Secure Admin Login
- JWT Authentication
- Protected Admin Routes
- CRUD Operations
- Resume Download
- GitHub & LinkedIn Integration
- Clean and Modern Design

# ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/BHAVESH0704/BhaveshPund-Frontend.git
```

### Navigate to the project

```bash
cd BhaveshPund-Frontend/client
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

---

# 🏗️ Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

---

# 🌍 Environment Variable

Create a `.env` file in the project root.

```env
VITE_API_BASE_URL=https://bhaveshpund-backend-production.up.railway.app/api/v1
```
